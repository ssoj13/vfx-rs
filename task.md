
Это порт _ref/OpenColorIO, _ref/OpenImageIO в vfx-rs, единую стройную систему предназначенную для работы в VFX/Image pipelines.
Крейт призван быть полной имплементацией API и всех фич и серьёзной работой облегчающей жизнь всем.
Так как OIIO/OCIO - open source, то мы просто берём их как reference code и используем в своём порте для построения новой системы без недостатков C++.
