Этот проект - порт C++ библиотек OpenImageIO / OpenColorIO на Rust. Нужно проверить совпадение всех важных функций API, Python API. Нужно корректным путём реализовать все функции, консультируясь с референсным кодом в _ref. Найти все нелогичные места и возможности для оптимизации, в том числе Rust-specific. Где можно используй готовые крейты если они подходящего качества. У нас есть улучшения over OIIO/OCIO, типа вьювера, других форматов файлов и всего такого - их тоже нужно отполировать и довести до высокого продакшн качества. Новые форматы файлов кроме тех что уже есть делать не надо. Используй sequential thinking and memory mcp, используй filesystem mcp, в filesystem есть множество инструментов. Сделай для себя FINDINGS.md и следуй ему, помечая галочками сделанное. Не пропускай ничего. Do not assume, do not guess, check everything.
